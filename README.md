## 取り付けイメージ  
### ノーマル版
VESA規格(75mm x 75mm)の穴に対応します。  
M4ネジとナットが別途必要です。  
キーボードの取り付けには100円均一などで買える耐震シートを使用します。  
写真では1.8mm厚の耐震シート(ロールになって売っていた)を使用しています。  
![](README/2025-11-27-20-03-02.png)  

キーボードを取り付けるとこんな感じ。  
耐震シートはかなり強くくっついてくれるのでちょっとやそっとじゃずれません。  
(※キーボードが落ちても責任はとれません。自己責任でお使いください。)  
耐震シートですが真横から引き延ばすように引っ張ると簡単に剥がれてくれます。  
![](README/2025-11-27-20-06-18.png)

### リストレストあり版
リストレストあり版も作成しました。  
手元に穴が2か所空いています。  
ここに別途印刷したリストレストを、これまた別途印刷した留め具で取り付けます。  
![](README/2025-11-27-20-07-35.png)  
![](README/2025-11-27-20-08-33.png)  

やや大きめのキーボードなどを使用しており固定が不安な人向けに面積が広い版も作成しました。  
![](README/2025-11-27-20-08-46.png)

使い方は同じです。  
![](README/2025-11-27-20-10-11.png)

### もっと大きい版
トラックボールマウスを横に置いておきたい人向けに、さらに大きい版も作成しました。  
![](README/2025-11-27-20-10-26.png)  
もっと大きい版は安定性に不安があったので、背面に金属製の拡張プレートを使用しています。ご参考まで。  
![](README/2025-11-27-20-22-13.png)  
使い方は同じです  
![](README/2025-11-27-20-11-06.png)

## 参考情報：使用している椅子とモニターアーム(構成例)
### 椅子
ニトリ：フォレスト

### モニターアーム
サンワサプライ：CR-LA360 を使っていますが、どうやら終売のよう。  
サンワサプライ：CR-LA358 が似ています。  
モニターアームを固定する部分の太さΦに差があるようです。
(※CR-LA360よりもCR-LA358の方が細い。)
